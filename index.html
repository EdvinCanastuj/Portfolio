Login HTML

El proyecto consiste en la implementación de un formulario de inicio de sesión utilizando las librerías de Bootstrap en un proyecto HTML. Bootstrap es un framework de desarrollo web que ofrece una amplia gama de componentes y estilos predefinidos, lo que facilita la creación de interfaces de usuario modernas y responsivas.

En este proyecto, se aprovecharán las ventajas de Bootstrap para diseñar y desarrollar un formulario de inicio de sesión atractivo y funcional. Se utilizarán las clases y estilos de Bootstrap para establecer la estructura del formulario, alinear los elementos correctamente y plicar estilos visualmente agradables

Api Rest Node.js Express

El proyecto consiste en la creación de una API REST utilizando Node.js y Express, dos tecnologías ampliamente utilizadas en el desarrollo de aplicaciones web y servicios API. La API permitirá realizar operaciones CRUD (Crear, Leer, Actualizar y Eliminar) sobre un conjunto de recursos, brindando una interfaz para que los clientes puedan interactuar y acceder a los datos de manera segura y eficiente.
